# 管理页统一下拉组件 Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** 将管理页当前 6 处原生下拉统一替换为可控的 `ui-select` 组件。

**Architecture:** 保持 Go + Gin + HTMX 的服务端渲染结构，在 `ui_htmx.go` 中新增通用下拉 helper 输出隐藏域和菜单结构，在 `htmx.js` 中以事件委托方式处理开合、选中和自动提交，在 `style.css` 中提供统一的 popover 样式和动画。表单提交字段名保持不变，因此后端业务处理不需要调整。

**Tech Stack:** Go 1.25, Gin, HTMX, 原生 JavaScript, CSS
